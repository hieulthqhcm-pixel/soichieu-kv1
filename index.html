Chưa có dữ liệu

📱 Hướng dẫn dùng trên iPhone

🚀 Cách nhanh nhất — Nút "Xuất App → iPhone"
  1. Trên máy tính: import 2 file Excel như bình thường
  2. Nhấn nút 📱 Xuất App → iPhone ở cột trái
  3. Máy tính tải về 1 file SoiChieu_KV1_[ngày].html
  4. Gửi file đó sang iPhone qua Zalo / AirDrop / iMessage / Email
  5. Trên iPhone: nhận file → mở bằng Safari → dữ liệu load tự động!
✅ File xuất ra có toàn bộ dữ liệu nhúng sẵn bên trong — không cần import lại trên iPhone
🔎 Tra cứu số container trên iPhone
  1. Mở file .html bằng Safari
  2. Cuộn xuống phần Tra Cứu Container (cột trái)
  3. Nhập số cont → kết quả hiện ngay: trạng thái soi, ngày giờ, người nhập
💾 Lưu app lên màn hình iPhone (dùng như app thật)
  1. Mở file trên Safari
  2. Nhấn biểu tượng Chia sẻ 〔hình vuông ↑〕ở thanh dưới
  3. Chọn "Thêm vào Màn hình chính"
  4. Đặt tên → nhấn Thêm → icon app xuất hiện trên màn hình
📲 Chia sẻ cho nhiều iPhone khác
Mỗi lần cập nhật dữ liệu mới → nhấn Xuất App → iPhone một lần → gửi file mới cho tất cả thành viên qua nhóm Zalo / group chat. Mọi người mở file mới là có dữ liệu mới nhất.
⚠️ Lưu ý: iPhone chỉ dùng để tra cứu và xem báo cáo. Import file Excel và tạo báo cáo vẫn cần làm trên máy tính.
